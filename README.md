# HACOM
부산대학교 정보컴퓨터공학과 2022 전기 졸업과제 (해컴)

## 프로젝트 개요
* 프로젝트 목적
* 적용된 기술
* 프로젝트 시작방법
* 저작권, 라이센스 정보


## 프로젝트 목적
기존에 존재하는 어류 판별이라는 서비스를 확장하여 어류 판별 뿐만 아니라 어류의 무게 추정을 통해 실생활의 편리함을 제공해주기 위해 프로젝트가 기획되었다.
